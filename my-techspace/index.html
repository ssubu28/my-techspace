Tech Projects

This page contains the various tech projects I have worked on during and post college.

Location based Query and Data Aggregation using RPL in ContikiOS -

Modeled the routing mechanism based on location hierarchy and compared its performance against existing implementation in ContikiOS. The modeled mechanism yielded better performance when sub-branches did not interfere.

Modification of the objective function in RPL and simulation of the graph using Cooja Simulator -

Allowed for packet forwarding across network by modifying the objective function of RPL protocol to suit our needs and simulating it in Cooja, a simulator used in Contiki-OS.

Doubly Linked circular list in C -

Implemented a doubly linked circular list and used it to store and process bank transactions.

Building a website -

Built a website containing 4 pages. Each page was linked to the other. Used various Bootstrap5 components, Google Fonts, custom stylesheets, images and many more.
